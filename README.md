# 🚀 JOLUX

> **Illuminate Learning**

JOLUX é uma plataforma de Inteligência Artificial criada para transformar a forma como professores planejam, ensinam e avaliam.

## 🎯 Missão

Ajudar professores a ensinar melhor através da inteligência artificial.

## 🚧 Status

Em desenvolvimento.

## 🛠️ Tecnologias

- Next.js
- React
- TypeScript
- FastAPI
- PostgreSQL
- Tailwind CSS
- OpenAI API
